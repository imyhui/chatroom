# 基于WebSocket的聊天室

> 操作系统课设，基于Python flask_socketio 及 WebSocket的多人聊天室

## 使用

```
git clone https://github.com/imyhui/chatroom.git
pipenv install
pipenv shell
python app.py
```

访问 http://0.0.0.0:5000/
